# 심리 성향 예측 AI 경진대회

1. 주제

심리학 테스트 분석 알고리즘 개발
 
2. 배경

심리학 테스트의 범주가 넓어짐에 따라 해당 영역의 데이터 분석 방법 탐구
국가 선거 투표자/미투표자의 심리학적 성향을 분석

3.대회 설명

마키아벨리즘 심리테스트를 활용하여 테스트 참가자의 국가 선거 투표 여부 예측
